Introduction: CREST ETL retrieves, process, and publishes the data necessary for the CREST modeling tool which integrates NASA Tropical Rainfall Measuring Mission (TRMM) satellite data and local rainfall information, taking into account evaporation, transpiration, soil porosity, soil moisture, and land shape/elevation to produce streamflow estimates.

Instructions to schedule the task on a machine:
1. Enter your credentials in config.pkl and config_CREST.xml
2. Save the file
3. Search for Administrative tools in your machine
4. Go to Task Scheduler
5. Click on "Create Task" in the right pane and enter details
6. Schedule the task in Triggers tab
7. Upload the CREST_ETL.bat file in the Actions tab
